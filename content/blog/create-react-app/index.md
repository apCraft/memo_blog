PWAに対応したreactを作成したい

JS用
> npx create-react-app my-pwa --template cra-template-pwa

typescript用
> npx create-react-app my-pwa --template pwa-typescript

ant design
> cd my-app/
> npm install --save antd
> npm install --save react-router-dom
> yarn start
